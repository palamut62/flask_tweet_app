🎯 CSS Selector Bulma Kılavuzu

1. Tarayıcı Developer Tools ile Selector Bulma

Adım 1: Developer Tools'u Açın

F12 tuşuna basın veya sağ tıklayıp "İncele" seçin

Adım 2: Element'i Seçin

Makale başlığına sağ tıklayıp "Inspect Element" seçin

Adım 3: Selector'ı Kopyalayın

Seçilen element üzerinde sağ tıklayıp:

  • Copy → Copy selector (Chrome/Edge)
  • Copy → CSS Selector (Firefox)

2. CSS Class Türleri ve Kullanımı

Tek Class

.article-title

Tek bir CSS class'ı seçer

Birden Fazla Class

.article-item.featured

Hem "article-item" hem de "featured" class'ına sahip elementleri seçer

Alt Element Seçimi

.article-container .title

"article-container" class'ı içindeki "title" class'ına sahip elementleri seçer

Direkt Alt Element

.article-list > .article-item

"article-list" class'ının direkt alt elementlerinden "article-item" class'ına sahip olanları seçer

4. Selector Optimizasyonu

💡 İyi Selector Özellikleri

  • Kısa ve öz: .article h2 yerine .article-title
  • Stabil: Sayfa güncellemelerinde değişmeyen class'lar
  • Benzersiz: Sadece istenen elementleri seçen
  • Genel: Tüm makaleler için çalışan

⚠️ Kaçınılması Gerekenler

  • Çok uzun selector'lar: body > div > div > div > article
  • ID'ler: #article-123 (her makale için farklı)
  • Pozisyon bazlı: :nth-child(3) (değişken)
  • Inline style'lar: [style="color: red"]

5. Test ve Doğrulama

Console'da Test Etme

Developer Tools'da Console sekmesine geçin ve şu komutu çalıştırın:

document.querySelectorAll('.your-selector')

Bu komut selector'ınızın kaç element bulduğunu gösterir.

Uygulama İçi Test

Haber kaynağı formunda "Manuel Selector'ları Test Et" butonunu kullanın.

6. Yaygın Sorunlar ve Çözümleri

Problem: Hiç Element Bulunamıyor

  • Selector'da yazım hatası var mı kontrol edin
  • Class adında nokta (.) var mı kontrol edin
  • Element dinamik olarak yükleniyor olabilir

Problem: Çok Fazla Element Buluyor

  • Daha spesifik bir selector kullanın
  • Parent element ile birleştirin
  • Ek class'lar ekleyin

Problem: Linkler Çalışmıyor

  • Base URL'yi doğru ayarlayın
  • Relative linkler için domain ekleyin
  • href attribute'unu kontrol edin

7. Pratik Örnekler

Örnek 1: Blog Sitesi

HTML Yapısı:

<article class="post">
  <h2 class="post-title">
    <a href="/article-url">Makale Başlığı</a>
  </h2>
  <div class="post-meta">
    <time class="post-date">2025-01-08</time>
  </div>
  <p class="post-excerpt">Makale özeti...</p>
</article>

Selector'lar:

Konteyner: .post
Başlık: .post-title a
Link: .post-title a
Tarih: .post-date
Özet: .post-excerpt

Örnek 2: Haber Sitesi

HTML Yapısı:

<div class="news-item featured">
  <h3>
    <a href="/news/123" class="news-link">Haber Başlığı</a>
  </h3>
  <span class="news-date">8 Ocak 2025</span>
  <div class="news-summary">Haber özeti...</div>
</div>

Selector'lar:

Konteyner: .news-item
Başlık: h3 .news-link
Link: .news-link
Tarih: .news-date
Özet: .news-summary

8. Gelişmiş Selector Teknikleri

Attribute Selector'ları

a[href*="/article/"] /* href'inde "/article/" geçen linkler */
div[class*="post"] /* class'ında "post" geçen div'ler */
img[alt] /* alt attribute'u olan resimler */

Pseudo-class'lar

.article:first-child /* İlk makale */
.article:last-child /* Son makale */
.article:nth-child(2) /* İkinci makale */
a:not(.external) /* external class'ı olmayan linkler */

Kombinasyonlar

.article h2, .article h3 /* h2 VEYA h3 */
.article.featured /* article VE featured class'ı */
.article > .title /* article'ın direkt alt elementi title */

Bu kılavuzu kullanarak herhangi bir haber sitesi için doğru selector'ları bulabilirsiniz!
